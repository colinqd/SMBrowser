# SMB文件管理器 - 功能简介与使用说明

## 软件简介

SMB文件管理器是一款基于Python开发的Windows平台SMB/CIFS网络文件管理工具。采用双栏界面设计，左侧为本地文件浏览器，右侧为远程SMB服务器文件浏览器，支持拖拽操作，提供直观的文件传输体验。

### 主要特性

- **双栏界面**：同时显示本地和远程文件系统，便于对比和操作
- **拖拽传输**：支持本地与远程之间的拖拽文件传输
- **安全加密**：配置文件全加密存储，主密码保护
- **多服务器管理**：支持保存多个服务器连接配置
- **实时进度**：文件传输显示进度百分比和传输速度
- **目录树导航**：树形结构快速浏览目录层级

## 功能详解

### 1. 连接管理

#### 1.1 首次使用
首次启动程序时，需要设置主密码（至少6个字符）。主密码用于加密保存的服务器配置信息。

#### 1.2 连接服务器
1. 点击工具栏「连接服务器」按钮
2. 在连接对话框中填写：
   - **服务器地址**：SMB服务器的IP地址或域名
   - **端口**：默认445，可根据服务器配置修改
   - **用户名**：SMB登录用户名
   - **密码**：SMB登录密码
   - **共享名称**：要访问的共享文件夹名称
   - **SMB版本**：支持SMBv1/v2/v3或自动协商
3. 点击「连接」按钮建立连接

#### 1.3 保存配置
在连接对话框中填写信息后，点击「保存配置」按钮，输入配置名称即可保存。下次使用时可从下拉列表快速选择。

#### 1.4 断开连接
点击工具栏「断开连接」按钮即可断开当前服务器连接。

### 2. 文件浏览

#### 2.1 本地文件浏览
- **目录树**：左侧显示本地磁盘和目录树结构
- **文件列表**：中下方显示当前目录的文件和子目录
- **导航**：双击目录进入，点击地址栏可输入路径

#### 2.2 远程文件浏览
- **服务器树**：连接后显示服务器共享列表
- **目录树**：展开共享显示目录结构
- **文件列表**：右下方显示远程文件列表

#### 2.3 排序功能
点击文件列表的列标题可按该列排序：
- 名称、大小、修改时间、类型
- 再次点击切换升序/降序

### 3. 文件传输

#### 3.1 拖拽上传
1. 在本地文件列表中选择文件或目录
2. 拖拽到远程文件列表区域
3. 文件将上传到当前远程目录

#### 3.2 拖拽下载
1. 在远程文件列表中选择文件或目录
2. 拖拽到本地文件列表区域
3. 文件将下载到当前本地目录

#### 3.3 传输进度
- 日志区域显示传输进度百分比（每2%更新）
- 显示实时传输速度（B/s、KB/s、MB/s）
- 显示文件总数和已完成数量

### 4. 安全管理

#### 4.1 主密码
- 用于加密保护服务器配置
- 首次使用需设置主密码
- 默认主密码：admin（建议修改）

#### 4.2 修改主密码
1. 菜单：设置 → 修改主密码
2. 输入新密码并确认
3. 修改后所有配置将使用新密码重新加密

#### 4.3 恢复默认密码
1. 菜单：设置 → 恢复默认主密码
2. 确认后将清空所有保存的配置
3. 主密码重置为默认值 admin

### 5. 配置文件加密

#### 5.1 加密机制
- 配置文件 `smb_config.dat` 采用全加密存储
- 使用 Fernet 对称加密算法
- 密钥通过 PBKDF2 从主密码派生
- 加密强度：100,000次迭代

#### 5.2 安全建议
- 设置强主密码（建议8位以上，包含字母数字符号）
- 不要使用默认主密码 admin
- 定期更换主密码
- 不要在公共电脑上保存敏感配置

## 系统要求

- **操作系统**：Windows 10/11
- **Python版本**：Python 3.8+（开发环境）
- **网络**：需能访问SMB服务器（端口445或自定义端口）
- **依赖库**：
  - pysmb（SMB协议支持）
  - cryptography（加密功能）
  - tkinterdnd2（拖拽支持）
  - Pillow（图标生成）

## 常见问题

### Q1: 无法连接到服务器
- 检查服务器地址和端口是否正确
- 确认用户名和密码无误
- 检查防火墙是否允许SMB端口
- 尝试更换SMB版本

### Q2: 忘记主密码怎么办
- 使用「恢复默认主密码」功能
- 注意：此操作会清空所有保存的配置

### Q3: 传输速度慢
- 检查网络连接质量
- 尝试使用SMBv3协议
- 减少同时传输的文件数量

### Q4: 配置文件在哪里
- 配置文件：`smb_config.dat`（加密存储）
- 盐值文件：`.smb_salt`（密钥派生用）
- 位于程序运行目录
