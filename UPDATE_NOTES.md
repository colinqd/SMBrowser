# SMB文件管理器 - 更新说明

## 版本 3.0 更新内容

### 架构重构
- **模块化设计**：将原单文件（smb_client.py，约1542行）重构为多模块架构
- **清晰的分层结构**：
  - `config/` - 配置管理
  - `core/` - 核心业务逻辑
  - `ui/` - 用户界面
- **更好的可维护性**：代码职责分离，易于扩展和维护

### 功能优化
- **扩展图标系统**：支持10+种文件类型（文档、表格、演示、PDF、图片、压缩包、音频、视频、代码等）
- **文件列表显示优化**：
  - 修复图标显示问题
  - 添加第一列（tree列）标题
  - 消除重复列显示
- **排序功能增强**：支持按名称（包含#0列）、大小、类型、修改时间排序

### 文件结构
```
pySmb/
├── main.py              # 主入口
├── config/
│   └── settings.py      # 配置管理
├── core/
│   ├── connection.py    # SMB连接管理
│   └── file_ops.py      # 文件操作
├── ui/
│   ├── main_window.py   # 主窗口
│   ├── dialogs/
│   │   └── connect_dialog.py
│   ├── utils/
│   │   ├── helpers.py   # 工具函数
│   │   └── icons.py     # 图标管理
│   └── widgets/
│       ├── file_panel.py
│       └── tree_view.py
└── requirements.txt     # 依赖包
```
